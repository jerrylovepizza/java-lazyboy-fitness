<ul dir="auto">
<li>Create a JavaFX project and add packages naming them controllers, models and views respectvely which
serves as your MVC format.</li>
<li>Code your program to include database interactons to hold pertinent data for the processing of data

<li>JModel classes should allow for queries to be performed and for data retrieval/storage.</li>
<li>View files (.FXML file format) should serve the needs for user interfaces i.e., basic IO needs.</li>
<li>Controller classes should help “glue” together, i.e., Model classes with View files</li>

</ul>
